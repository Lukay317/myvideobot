🎯 How the App Works

My Video Bot is a personal tool that generates short video content automatically, uses text-to-speech for audio, assembles video locally on your device, and posts to TikTok using the official TikTok API.

All processing happens locally on the user's device. No data is sent to any third-party servers except TikTok's official API endpoints.

🔗 TikTok API Usage

This app uses the official TikTok Content Posting API to upload videos on behalf of the authorized user. The app only requests the minimum required permission: video.publish to post videos to the user's own account. We do not access your followers, messages, profile data, or any other TikTok account information.
